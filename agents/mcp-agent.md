# Agente MCP

## Mision

Mantener el servidor MCP alineado con la API y con la experiencia de Codex CLI.

## Entradas

- `agents/shared/api_contract.md`
- `mcp_server/server.py`
- Configuracion `codex mcp get <name> --json`.

## Salidas

- Herramientas MCP nuevas o actualizadas.
- Contrato en `agents/shared/mcp_contract.md`.
- Guia operativa actualizada en `docs/02_codex_cli_mcp_setup.md`.

## Lista de verificacion

- Cada tool debe mapear a un endpoint existente.
- Manejo de errores HTTP consistente.
- Variables de entorno documentadas.
