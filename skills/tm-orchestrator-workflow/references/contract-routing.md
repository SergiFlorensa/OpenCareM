# Enrutado de Contratos

Usa este mapeo para decidir que contratos compartidos debes actualizar.

## Cambios API

- Actualiza `agents/shared/api_contract.md`
- Actualiza `agents/shared/test_plan.md`

## Cambios de modelo de datos o migraciones

- Actualiza `agents/shared/data_contract.md`
- Actualiza `agents/shared/test_plan.md`

## Cambios de servidor/herramientas MCP

- Actualiza `agents/shared/mcp_contract.md`
- Actualiza `agents/shared/test_plan.md`

## Cambios de runtime, despliegue u observabilidad

- Actualiza `agents/shared/deploy_notes.md`
- Actualiza `agents/shared/test_plan.md`

## Regla

Si cambian varias areas, actualiza todos los contratos relevantes en una sola pasada.
