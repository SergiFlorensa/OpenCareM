# Decisions Log (ADR)

Este directorio guarda decisiones de arquitectura y operacion.

## Formato recomendado

Crear un archivo por decision:

`docs/decisions/ADR-XXXX-titulo-corto.md`

Plantilla minima:

```md
# ADR-XXXX: Titulo

## Contexto
Problema y restricciones.

## Decision
Que se decide exactamente.

## Consecuencias
Impacto positivo y costo.

## Validacion
Como comprobar que funciona.
```
